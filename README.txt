To run this code, follow the steps below:
1.	Make sure you have Node.js and visual studio code installed on your machine. 
2.	Import the project folder from my GitHub repository:      
3.	Open the visual studio code navigates to File menu  Click on Open Folder.. and to selected the project folder (Imported in stpe2) and Accept the Trusted one
4.	Open the DataFiles folder and confirms that there is no output.json file
5.	Go to the top menu and Navigate to Terminal  New Terminal(Click) 
6.	Navigate to the directory where the CloudRay.js file is saved. And Run the following command to execute the script: node CloudRay.js
7.	Open the DataFiles folder and verify the output.json file for required data.
